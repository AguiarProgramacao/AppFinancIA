# Politica de Privacidade - FinancIA

Data de vigencia: 14/01/2026

Aguiar Programacao LTDA (CNPJ 54.819.426/0001-09) ("Aguiar Programacao", "nós") e a responsavel pelo aplicativo FinancIA. Esta politica descreve como tratamos dados pessoais no app, em conformidade com a LGPD (Lei 13.709/2018).

Contato: programacaoaguiar@gmail.com

## 1. Dados pessoais tratados
Coletamos e tratamos dados fornecidos pelo usuario e gerados pelo uso do app, incluindo:
- Cadastro e autenticacao: nome, e-mail e senha (tratada pelo backend).
- Perfil: remuneracao, foto de perfil (opcional).
- Finanças: transacoes (tipo, valor, descricao, data, categoria, forma de pagamento, observacoes).
- Objetivos: metas, economizado, prazo e aportes.
- Preferencias: configuracoes de notificacoes (push e e-mail).
- Identificadores tecnicos: token de sessao e token de notificacao push.
- Informacoes de dispositivo: nome/modelo do dispositivo e sistema operacional (enviado no login para identificacao de sessao).

## 2. Finalidades do tratamento
Os dados sao usados para:
- Autenticar e manter a sessao do usuario.
- Registrar e exibir transacoes, objetivos e graficos.
- Gerar o resumo do dashboard e insights financeiros.
- Enviar notificacoes relacionadas ao app (quando habilitado).
- Permitir exportacao de dados (CSV, JSON, PDF).
- Reforcar seguranca (2FA, gerenciamento de sessoes e biometria local).

## 3. Base legal (LGPD)
Tratamos dados com base em:
- Execucao de contrato: para operar o app e entregar suas funcionalidades.
- Consentimento: para notificacoes e recursos opcionais.
- Legitimo interesse: para seguranca e prevencao de uso indevido.

## 4. Compartilhamento de dados
Nao vendemos dados pessoais. O app integra com:
- Backend do FinancIA (API) para processar dados e autenticacao.
- Servicos do Expo para notificacoes push (quando habilitado).
- Servicos do sistema operacional para biometria local e selecao de imagens.

Nao ha compartilhamento para fins publicitarios.

## 5. Armazenamento e seguranca
- Dados locais: token de sessao, usuario, configuracoes e token de push sao armazenados no dispositivo via AsyncStorage.
- Medidas de seguranca incluem autenticacao, 2FA e controle de sessoes.

## 6. Retencao
Nao ha um periodo de retencao especificado no app. Os dados sao mantidos enquanto necessários para o funcionamento do FinancIA. Para esclarecimentos, contate o e-mail informado.

## 7. Direitos do titular
Voce pode solicitar informacoes, correcoes e esclarecimentos sobre seus dados pelo e-mail de contato.
No momento, o app nao oferece funcionalidade de exclusao de conta ou eliminacao de dados.

## 8. Criancas e adolescentes
O app nao e direcionado a criancas ou adolescentes e nao realiza tratamento intencional de dados de menores.

## 9. Alteracoes desta politica
Podemos atualizar esta politica periodicamente. A versao vigente sera disponibilizada junto ao app.
